Drag Menu Example

Drag the menu icon to the left to reveal the menu.

Additional Demos: Pull Right Demo | Pull Left Demo | Pull Down Demo

Code used to initialize this demo:

jQuery("#menu")
  .dragMenu(
	"#menu-drag-handle", 
	"#content", 
	{ direction: "left" }
  );
